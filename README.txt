DBCLientApp - The purpose of this program is to record and display customer information and the appointments associated with them.

Author - Brad Pappan, brad.pappan2@gmail.com, 4/29/2022

IDE Version - IntelliJ IDEA 2021.3 (Community Edition), Runtime version: 11.0.13+7-b1751.19 amd64, JavaFX-SDK-17.0.1

Once you have installed all the listed IDE and JDK's, you would have to set a MySQL database with the following user information listed in the JDBC class,
or adjust the data accordingly. The program can be ran after installing all of that.

The additional report I made was to count all listed customers in the database.

mysql-connector-java-8.0.25
